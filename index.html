Desmos Connect

Desmos Connect is an interactive version of Desmos made for Adobe Connect. It allows students and teachers to interact with a Desmos graphing calculator in real time.

Note: Desmos Connect does not support the Classic view in Adobe Connect.


Desmos Connect features:

  • Expressions
  • Notes
  • Import from URL
  • Participant Access
  • Works with Breakout Rooms
  • Open Source
